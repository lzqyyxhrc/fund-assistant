2. 选择日期范围: 2023-06-05 至 2026-06-04
3. 下载为CSV文件
4. 重命名为 `NDX_YYYYMMDD.csv` (YYYYMMDD为下载日期)

## 🔧 自定义回测

### 修改投资组合权重
编辑 `backtest.py` 中的权重配置:

```python
portfolio_weights = {
    "NDX": 0.4,      # 纳斯达克100 - 40%
    "512890": 0.4,   # 红利低波ETF前复权 - 40%
    "AU9999": 0.2    # 黄金 - 20%
}
```

### 添加新的指数
1. 下载新指数数据到 `data/index/`
2. 在脚本中修改 `INDEX_CONFIG`
3. 重新运行回测

### 回测时间范围
默认回测时间范围是最近3年。如需修改，编辑 `download_index_data.py` 中的 `start_date`:

```python
if start_date is None:
    start_date = (datetime.now() - timedelta(days=365*3)).strftime("%Y-%m-%d")
```

## 📈 回测指标说明

- **总收益率**: 投资期间的总回报
- **年化收益率**: 平均每年的复合增长率
- **年化波动率**: 收益的标准差，衡量风险
- **夏普比率**: 风险调整后的收益 (越高越好)
- **最大回撤**: 从最高点到最低点的最大跌幅

## ⚠️ 注意事项

1. **数据完整性**: 确保所有指数数据的时间范围一致
2. **汇率因素**: 纳斯达克100为美元计价，需要考虑汇率影响
3. **历史表现**: 过去的表现不代表未来的收益
4. **费用**: 回测未考虑交易费用和滑点
5. **流动性**: 实际交易中大额买卖可能影响价格
